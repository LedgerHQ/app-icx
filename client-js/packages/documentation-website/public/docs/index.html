▸ getAddress(path, boolDisplay, boolChaincode)

Returns public key and ICON address for a given BIP 32 path.

getAddress(path: string, boolDisplay: boolean, boolChaincode: boolean): Promise<{publicKey: string, address: string, chainCode: string?}>
Parameters
path (string) a path in BIP 32 format
boolDisplay (boolean = false)
boolChaincode (boolean = true)
Returns
Promise<{publicKey: string, address: string, chainCode: string?}>: an object with a publickey(hexa string), address(string) and (optionally) chaincode(hexa string)
Example
icx.getAddress("44'/4801368'/0'", true, true).then(o => o.address)
